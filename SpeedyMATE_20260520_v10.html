📖 Glosar

TermenDefiniție
CifrăSimbol pentru scrierea numerelor (0–9 în sistemul zecimal).
Număr naturalElement din N = {0, 1, 2, 3, ...}.
OrdinePoziția cifrelor: unități, zeci, sute, mii, zeci de mii, sute de mii, milioane.
ClasăGrup de 3 ordine: clasa unităților, clasa miilor, clasa milioanelor.
SuccesorNumărul imediat următor: succ(n) = n + 1.
PredecesorNumărul imediat anterior: pred(n) = n − 1.
Cifre romaneI=1, V=5, X=10, L=50, C=100, D=500, M=1000.
ComparareStabilirea relației: <, >, =.
Ordonare crescătoareDe la cel mai mic la cel mai mare.
Ordonare descrescătoareDe la cel mai mare la cel mai mic.
RotunjireAproximarea la un anumit ordin.
Aproximare prin lipsăCifra rămâne; cifrele inferioare → 0.
Aproximare prin adaosCifra crește cu 1; cifrele inferioare → 0.
Termeni (adunare)Numerele care se adună.
SumăRezultatul adunării.
Element neutru (+)a + 0 = a.
Comutativitatea + b = b + a.
Asociativitate(a+b)+c = a+(b+c).
DescăzutNumărul din care se scade.
ScăzătorNumărul care se scade.
DiferențăRezultatul scăderii.
Proba scăderiidiferență + scăzător = descăzut.
Putereaⁿ = a × a × ... × a (n factori).
BazăNumărul care se înmulțește repetat.
ExponentDe câte ori se înmulțește baza.
Pătrat perfectn² pentru n natural: 1, 4, 9, 16, 25...
